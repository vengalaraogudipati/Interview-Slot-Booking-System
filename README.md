# Interview Slot Booking System

## About The Project

This is a Full Stack Interview Slot Booking System developed using React JS, Spring Boot, Spring Security, and MySQL.

The main purpose of this project is to simplify interview scheduling between candidates and admin. Users can register, login, view available interview slots, and book slots. Admin can manage interview slots, approve or reject bookings, and monitor the complete booking process.

This project helped me improve my skills in Java Full Stack Development, REST APIs, Spring Security, React Routing, and Database Management.
